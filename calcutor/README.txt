# Calcutor
A TI-83 Emulator.  Available on calcutor.net(http://calcutor.net).

## Collaborators:

## IMPORTANT

**Make sure to run `python setup.py develop` after pulling
down from GitHub.  This will create the system-specific egg
for gunicorn, etc. to use**
